## 集成

### 使用 CocoaPods
1. 终端 cd 至项目根目录
2. 执行 pod init
3. 执行 open -e Podfile
4. 添加导入配置 pod 'RCSceneVideoRoom'
5. 执行 pod install
6. 双击打开 .xcworkspace
